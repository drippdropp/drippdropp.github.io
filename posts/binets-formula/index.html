Binet's Formula

An extremelly efficient method for generating the nnnth value of the Fibonacci sequence is given by the formula,

fn=Φn−(−ϕ)n5 , \boxed{f_n = \frac{\Phi^n - (-\phi)^n}{\sqrt{5}}}\,, fn​=5​Φn−(−ϕ)n​​,

where Φ\PhiΦ is the golden ratio and −ϕ-\phi−ϕ is the golden ratio conjugate.

Every fkf_kfk​ where k≡0(mod3)k \equiv 0 \pmod{3}k≡0(mod3) is even and every fkf_kfk​ where k≡{1,2}(mod3)k \equiv \{ 1, 2 \} \pmod{3}k≡{1,2}(mod3) is odd.

The tricky thing is that when we write an implementation of this function, we need to be mindful that the golden ratio is a transcendental number meaning that it has infinitely many digits like π\piπ. We leverage Julia's BigFloat and BigInt types to ensure we use enough digits to reduce rounding errors. Regardless, we'll still get plenty of rounding errors and as nnn approaches infinity, we will increasely get more results which need to be rounded down slightly. To combat this, after performing the computation, we round the result and then convert from BigFloat to BigInt.

Φ, ϕ = big(MathConstants.golden), big(MathConstants.golden) - 1.0
fib(n) = (Φ^n - (-ϕ)^n)/sqrt(5) |> round |> BigInt
