Euler's Totient Function

The totient function, ϕ(n)\phi(n)ϕ(n), is defined as the number of positive integers up to nnn which are relatively prime to nnn.

For example, ϕ(10)=4\phi(10) = 4ϕ(10)=4 because for the interval [1,…,10][1,\dots,10][1,…,10], the numbers 111, 333, 777, and 999 are relatively prime to 101010.

Therefore, it turns out that if nnn is prime, then ϕ(n)=n−1\phi(n) = n - 1ϕ(n)=n−1.

If ppp is a prime and aaa is a positive integer, then,

ϕ(pa)=pa−pa−1 . \phi(p^a) = p^a - p^{a-1} \,.ϕ(pa)=pa−pa−1.

If aaa and bbb are relatively prime and n=abn = abn=ab, then ϕ(n)=ϕ(a)ϕ(b)\phi(n) = \phi(a)\phi(b)ϕ(n)=ϕ(a)ϕ(b).
