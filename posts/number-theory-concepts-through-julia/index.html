Number Theory Concepts through Julia

We start with the digital root, which is defined as the sum of all the digits of a number, nnn.

More concretely, suppose that,

n=d0+d1b+d2b2+⋯+dnbn ,n = d_0 + d_1 b + d_2 b^2 + \cdots + d_n b^n\, ,n=d0​+d1​b+d2​b2+⋯+dn​bn,

therefore the digits are given as {d0,d1,d2,…,dn}\{ d_0, d_1, d_2, \dots, d_n \}{d0​,d1​,d2​,…,dn​} and so the digital root, DDD, is defined by,

D≡∑i=0ndiD \equiv \sum_{i=0}^n d_i D≡i=0∑n​di​ 
function digitalroot(n::Int)
    while (n = sum(digits(n))) > 9 end
    return n
end

or using recursion,

function digitalroot(n::Int)
    n < 10 && return n
    return digitalroot(n |> digits |> sum)
end

We are often interested in the divisors of a number, nnn,

function divisors(n::Int)
    n == one(n) && return [one(n)]
    ds = [ 1, n ]
    
    i = 2
    while i ≤ sqrt(n)
        if n % i == 0
            if i != div(n, i)
                ds = vcat(ds, [i, div(n, i)])
            else
                ds = vcat(ds, [i])
            end
        end
        i += 1
    end
    return sort(ds)
end

Right-to-Left Exponentiation

function rtlexp(x, n)
    y = 1
    while n > 0
        if isodd(n)
            y = x*y
        end
        x = x * x
        n = fld(n, 2)
    end
    y
end
