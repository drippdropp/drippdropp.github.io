Computing the Modular Inverse

When we're doing modular arithmetic, we need to use a different type of inverse to isolate terms. Suppose we have the equation,

17x≡1(mod5) . 17x \equiv 1 \pmod{5} \,.17x≡1(mod5).

If this were not in modulo 555, we would simply divide both sides by 171717 yielding the solution of xxx to be 117\frac{1}{17}171​. Unfortunately, that won't work for us this time, so we need to find the value of xxx such that the product of 171717 and xxx yields 111 modulo 555.

In Julia, we have a function which comes with the standard library, invmod which we can simply pass in 171717 and 555 and get a result.

To get a better sense of what is happening under the hood, we'll handroll an implementation of invmod.

function myinvmod(a, m)
    d, x, _ = gcdx(a, m)
    d > 1 && throw(MethodError("Modular Inverse does not exist"))
    
    mod(x, m)
end

We use the gcdx function which is the extended Euclidean algorithm which returns the gcd⁡(a,b)\gcd(a, b)gcd(a,b), the initial xxx value and the initial yyy value. If gcd⁡(a,b)≠1\gcd(a, b) \neq 1gcd(a,b)=1, then there is no modular inverse. Otherwise, we return xmod  mx \mod{m}xmodm.

Example

We want to find 14−1(mod17)14^{-1} \pmod{17}14−1(mod17), so we're looking for,

14x≡1(mod17) , 14x \equiv 1 \pmod{17}\,, 14x≡1(mod17),

which we can write as,

14X−17Y=1 . 14X - 17Y = 1 \,.14X−17Y=1.

It's pretty obvious that gcd⁡(14,17)=1\gcd(14, 17) = 1gcd(14,17)=1, but let's verify it.

17=14⋅1+314=3⋅4+23=2⋅1+12=1⋅2+0\begin{aligned} 17 &= 14 \cdot 1 + 3 \\ 14 &= 3 \cdot 4 + 2 \\ 3 &= 2 \cdot 1 + \boxed{1} \\ 2 &= 1 \cdot 2 + 0 \end{aligned}171432​=14⋅1+3=3⋅4+2=2⋅1+1​=1⋅2+0​

Working backwards now. We set a=17a = 17a=17 and b=14b = 14b=14.

3=a−1⋅bb=(a−b)⋅4+22=b−(a−b)⋅41=a−1⋅b−(b−4(a−b))\begin{aligned} 3 &= a - 1 \cdot b \\ \\ b &= (a - b)\cdot 4 + 2 \\ 2 &= b - (a - b)\cdot 4 \\ 1 &= a - 1 \cdot b - (b - 4(a - b)) \end{aligned}3b21​=a−1⋅b=(a−b)⋅4+2=b−(a−b)⋅4=a−1⋅b−(b−4(a−b))​

Simplifying the last line,

1=a−b−b+4a−4b=5a−6b ,1 = a - b - b + 4a - 4b = 5a - 6b\,,1=a−b−b+4a−4b=5a−6b,

which yields our final solution, 14(−6)+17(5)=114(-6) + 17(5) = 114(−6)+17(5)=1 implying that X=−6X = -6X=−6 and Y=5Y = 5Y=5.

Finally, we want to compute −6≡x(mod17)-6 \equiv x \pmod{17}−6≡x(mod17) which in this case is, −6+17=11-6 + 17 = 11−6+17=11.

We can verify this is the case as well by plugging 111111 back into our original equation to determine if 111111 is the modular inverse of 141414 modulo 171717. In fact, 14⋅11=15414 \cdot 11 = 15414⋅11=154 and since 17⋅9=15317\cdot 9 = 15317⋅9=153, we have indeed found our inverse.
