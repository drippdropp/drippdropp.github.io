The Area of a Triangle

There are several ways to compute the area of a triangle depending on what information is available.

For the purposes of method dispatch, we'll define a wrapper for our angles. Otherwise we'll have some ambiguity with respect to a function with three float inputs. This will make sense as you read on.

struct Angle
    θ::Float64
end

For any triangle, using the base, bbb, and height, hhh,

A=12⋅b⋅h . A = \frac{1}{2}\cdot b \cdot h \,.A=21​⋅b⋅h. 
A(base::T, height::T) where {T<:Real} = 0.5 * base * height

Using the sides, aaa, bbb, and ccc, where s=12(a+b+c)s = \frac{1}{2}(a + b + c)s=21​(a+b+c),

A=s(s−a)(s−b)(s−c) . A = \sqrt{s ( s - a )(s - b)(s - c)}\,.A=s(s−a)(s−b)(s−c)​. 
function A(a::T, b::T, c::T) where {T<:Real}
    s = (a + b + c)/2
    sqrt(s * (s - a) * (s - b) * (s - c))
end

Using the angle, θ\thetaθ, formed by two sides aaa and bbb,

A=12absin⁡θ . A = \frac{1}{2} a b \sin \theta \,.A=21​absinθ. 
A(a::T, b::T, θ::Angle) where {T<:Real} = 0.5 * a * b * sin(θ.θ)

Suppose we have a triangle which has sides of length 888, 888, and 121212.

Since we don't have the height, we need to compute it using the Pythagorean theorem as h=82−(122)2h = \sqrt{8^2 - \Big(\frac{12}{2}\Big)^2}h=82−(212​)2​.

# Using 1/2 * base * height
A(12.0, sqrt(8^2 - (12/2)^2))
> 31.74901573277509

Using Heron's formula, we just plug in the value from each side.

A(8, 8, 12)
> 31.74901573277509

Using two sides and the angle formed between them. We'll need to compute the angle. We use the trigonometric relation b/2=acos⁡(θ)b/2 = a\cos(\theta)b/2=acos(θ) resulting in θ=arccos⁡b2a\theta = \arccos{\frac{b}{2a}}θ=arccos2ab​.

A(8, 12, Angle(acos(6/8)))
> 31.74901573277509

Three different formulas, one result.
