Norms

Norms are a generalization of a length function and ultimately are used for computing the length of vectors in an abstract vector space.

Let VVV be an abstract vector space over a field F\mathbb{F}F. The norm is a function, η:V→R\eta : V \rightarrow \mathbb{R}η:V→R such that,

  1. η(x+y)≤η(x)+η(y)\eta(x + y) \leq \eta(x) + \eta(y)η(x+y)≤η(x)+η(y) for all x,yx, yx,y in VVV.

  2. η(αx)=∣α∣η(x)\eta(\alpha x) = \vert \alpha \vert \eta(x)η(αx)=∣α∣η(x) for all xxx in VVV and α\alphaα in F\mathbb{F}F.

  3. For all xxx in VVV, η(x)=0\eta(x) = 0η(x)=0 if and only if x=0x = 0x=0.

For each of the following norms I discuss below, I've provided an implementation in Julia. That being said, there's a great package Distances which includes all of these and more. While my implementations are good for solidifying the concepts behind the individual functions, they are likely not as efficient as those defined in Distances.

Absolute Value Norm

The absolute value function,

η(x)=∣x∣ . \eta(x) = \vert x \vert\,. η(x)=∣x∣.

In 111-dimensional space, the vector which extends from the origin to ±x\pm x±x has a length of xxx.

The vector ±4\pm 4±4 has a length of 444.

ℓ1\ell^1ℓ1 Norm

The ℓ1\ell^1ℓ1 norm will trace along the individual components of a vector rather than calculate the distance along the hypotenuse,

η(x)=∑i=1n∣xi∣ . \eta(x) = \sum_{i=1}^n \vert x_i \vert \,.η(x)=i=1∑n​∣xi​∣.

Therefore, by the ℓ1\ell^1ℓ1 norm, the (4,3)(4, 3)(4,3) vector has a length of 4+3=74 + 3 = 74+3=7.

l1_norm(X) = [abs(x) for x in X] |> sum

l1_norm([4 ; 3])
> 7

ℓ2\ell^2ℓ2 Norm

The ℓ2\ell^2ℓ2 norm is for computing the length of a vector in nnn-dimensional space, defined as,

η(x)=∥x∥=x12+⋯+xn2 . \eta(x) = \lVert \mathbf{x} \rVert = \sqrt{x_1^2 + \cdots + x_n^2 }\,. η(x)=∥x∥=x12​+⋯+xn2​​.

Notice that this is the same as the square root of the inner product of a vector with itself, x⋅x\sqrt{\mathbf{x}\boldsymbol{\cdot} \mathbf{x}}x⋅x​.

By the ℓ2\ell^2ℓ2 norm, the vector (4,3)(4, 3)(4,3) has a length of 32+42=9+16=25=5\sqrt{3^2 + 4^2} = \sqrt{9 + 16} = \sqrt{25} = 532+42​=9+16​=25​=5.

l2_norm(X) = [x*x for x in X] |> sum |> sqrt

l2_norm([4 ; 3])
> 5.0

ℓp\ell^pℓp Norm

The ℓp\ell^pℓp norm is not actually a single norm, but a generalization of multiple norms.

η(x)=∥x∥p=(∑i=1n∣xi∣p)1/p .\eta(x) = \lVert \mathbf{x} \rVert_p = \Big( \sum_{i=1}^n \vert x_i \vert^p \Big)^{1/p}\,.η(x)=∥x∥p​=(i=1∑n​∣xi​∣p)1/p.

It's trivial to see that when p=1p = 1p=1, we get the ℓ1\ell^1ℓ1 norm, when p=2p = 2p=2, we get the ℓ2\ell^2ℓ2 norm, and so on. When ppp approaches ∞\infty∞, then we get the infinity norm which is defined as,

η(x)=∥x∥∞=max⁡i∣xi∣ .\eta(x) = \lVert \mathbf{x} \rVert_\infty = \max_i \vert x_i \vert \,.η(x)=∥x∥∞​=imax​∣xi​∣. 
l∞_norm(X) = [abs(x) for x in X] |> maximum

l∞_norm([4 ; 3])
> 4

Therefore, we can write a generalized function to handle any ℓp\ell^pℓp norm as follows.

function lp_norm(X; p = 1)
    if isinf(p)
        [abs(x) for x in X] |> maximum
    else
        ([abs(x)^p for x in X] |> sum)^(1/p)
    end
end

@testset "ℓp Norms" begin
    X = [ 4 ; 3 ]
    @test lp_norm(X) == 7.0
    @test lp_norm(X; p=2) == 5.0
    @test lp_norm(X; p=Inf) == 4.0
end
