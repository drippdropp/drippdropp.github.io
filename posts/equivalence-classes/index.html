Equivalence Classes

Given a set, SSS and an equivalence relation, ∼\sim∼, the equivalence class, α\alphaα, is defined as,

{x∈S∣x∼α} . \{ x \in S | x \sim \alpha \}\,.{x∈S∣x∼α}.

An equivalence relation on a set XXX must satisfy three properties.

  1. α∼α\alpha \sim \alphaα∼α for all α\alphaα in XXX (reflexivity)

  2. α∼β\alpha \sim \betaα∼β implies β∼α\beta \sim \alphaβ∼α for all α\alphaα and β\betaβ in XXX (symmetry)

  3. If α∼β\alpha \sim \betaα∼β and β∼γ\beta \sim \gammaβ∼γ, then α∼γ\alpha \sim \gammaα∼γ for all α\alphaα, β\betaβ, and γ\gammaγ in XXX (transitivity)

function generate_equivclasses(N; S)
    X = Array{Set{eltype(S)}, 1}(undef, N)
    for k in [0, 1, 2]
        X[k+1] = Set(filter(x -> mod(x, N) == k, S))
    end
    return X
end

N = 3;
S = collect(-15:15);

X = reduce((iter, el) -> union(iter, el),
